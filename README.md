Amazon Mart Sales Analysis Dashboard

An interactive Power BI Sales Analytics Dashboard developed to monitor and evaluate Amazon Mart's business performance through data-driven insights. The dashboard consolidates key sales metrics, customer segmentation, regional performance, and product-level analysis into a single, intuitive reporting interface.

It provides stakeholders with a comprehensive view of business performance by highlighting sales trends, profitability, product demand, and customer behavior. Interactive filters and dynamic visualizations enable users to explore data across multiple dimensions, supporting informed decision-making and strategic planning.

Key Highlights
Developed an interactive dashboard using Power BI.
Monitored key performance indicators (KPIs) including Sales, Profit, Quantity Sold, and Discount.
Analyzed sales performance across Customer Segments, Regions, Categories, and Sub-Categories.
Identified top-performing products based on Sales, Profit, and Quantity Sold.
Enabled dynamic data exploration through interactive slicers and filters.
Utilized Power Query for data transformation and DAX for creating calculated measures and business metrics.
Designed a clean, user-friendly interface to facilitate efficient business reporting and performance analysis.

Technologies Used
Power BI
Power Query
Interactive Data Visualization

Project Objective
The primary objective of this dashboard is to provide a centralized reporting solution that enables business users to monitor sales performance, identify growth opportunities, evaluate product profitability, and support strategic decision-making through interactive and insightful visual analytics.
